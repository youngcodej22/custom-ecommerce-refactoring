# E-commerce with Custom CSS Framework 연습 - 기록

## 진행 순서

### 환경설정

#### packages

1. vite
    - 비트는 자바스크립트 네이티브 모듈 (opens new window)을 기반으로 한 데브 서버
    - `yarn create vite`
2. aos
    - Animate On Scroll library
    - `yarn add aos`
3. node-sass
    - node 환경에서 사용가능한 sass
    - `yarn add node-sass`
4. swiper
    - carousel library
    - `yarn add swiper`
5. sass
    - css preprocessor
    - `yarn add -D sass`
6. @types/aos
    - vite를 통해 설치 한 것 외에 추가 설치
